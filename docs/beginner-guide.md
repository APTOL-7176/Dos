## 🏃 첫 번째 런

### 던전 입장

**시작 위치**: 1층 첫 번째 방

**목표**: 5층 보스 처치

**준비물**:
- 기본 장비
- 포션 3개
- 요리 재료 없음

### 탐험 순서

**1. 주변 탐색**
```
방 구조 파악 → 적 위치 확인 → 안전한 루트 찾기
```

**2. 첫 전투**
```
약한 적 발견 → BRV 공격 3회 → HP 공격 1회 → 승리
```

**3. 채집**
```
채집 오브젝트 발견 → 상호작용 (E키) → 재료 획득
```

**4. 요리솥 찾기**
```
요리솥 (Ω) 발견 → 요리 제작 → 버프 적용
```

**5. 휴식**
```
안전 지대 → 휴식 (R키) → HP/MP 회복
```

**6. 다음 층**
```
내려가는 계단 (>) → 상호작용 → 2층 진입
```

## ⚔️ 전투 기초

### ATB 게이지 이해하기

**ATB란?**
- Active Time Battle (실시간 전투)
- 게이지 1000 도달 시 행동 가능
- 속도 스탯에 비례하여 증가

**행동 순서**:
```
ATB 충전 → 1000 도달 → 스킬 선택 → 실행 → ATB 감소
```

### BRV 시스템 이해하기

**BRV란?**
- Brave (용기 포인트)
- HP 데미지 주기 전 축적하는 중간 단계

**전투 흐름**:
```
BRV 공격 (Q) → BRV 축적 → HP 공격 (E) → HP 데미지
```

**예시**:
```
1턴: BRV 공격 → BRV 1000 → 1300 (+300)
2턴: BRV 공격 → BRV 1300 → 1600 (+300)
3턴: BRV 공격 → BRV 1600 → 1900 (+300)
4턴: HP 공격 → BRV 1900 → 100 (적 HP -475)
```

### 기본 전투 패턴

**초보자 안전 패턴**:
```
BRV 공격 x3 → HP 공격 x1 → 반복
```

**이유**:
- BRV 충분히 축적 → 높은 HP 데미지
- 안정적이고 예측 가능
- 실수 여지 적음

### 스킬 사용

**스킬 선택**:
- 1~6: 스킬 1~6
- Q: BRV 공격
- E: HP 공격
- R: 궁극기

**MP 관리**:
- 일반 전투: BRV/HP 공격 (MP 0)
- 위험 전투: 스킬 사용 (MP 소모)
- 보스전: 궁극기 (MP 50)

## 🛡️ 생존 팁

### 1. HP 관리

**회복 방법**:
1. **포션 사용** (I키 → 포션 선택)
2. **휴식** (안전 지대에서 R키)
3. **요리 섭취** (HP 회복 요리)
4. **치유의 샘** (맵에서 발견)

**회복 타이밍**:
- HP 50% 이하: 포션 사용
- HP 30% 이하: 즉시 회복!
- 전투 전: 100% 유지

### 2. MP 관리

**MP 절약**:
- 일반 전투: 기본 공격만
- 강한 적: 스킬 사용
- 보스: 궁극기

**MP 회복**:
- 포션
- 휴식
- MP 회복 요리
- 패시브 (마나 연결)

### 3. 상처 시스템

**상처란?**
- HP 데미지의 25%가 상처로 누적
- 최대 HP 감소 (일시적)
- 자연 회복 느림

**상처 치료**:
1. 휴식 (50% 치료)
2. 회복 아이템 (효율 1.5배)
3. 치유의 샘 (완전 치료)

**주의**:
- 상처 50% 누적 → 최대 HP 절반으로 감소
- 장기전 전 상처 치료 필수

### 4. 인벤토리 관리

**무게 제한**:
- 최대 무게: 100
- 초과 시 이동 속도 감소

**정리 팁**:
- 저가 아이템 버리기
- 중복 장비 판매
- 재료는 요리로 변환

### 5. 함정 회피

**함정 종류**:
- 스파이크 (데미지)
- 독 (상태 이상)
- 마법 (디버프)

**회피 방법**:
- 천천히 이동
- 지각 스탯 투자
- 해제 스킬

## 🍳 요리 활용

### 요리 기초

**요리솥 찾기**:
- 심볼: Ω
- 층마다 최소 1개 보장
- 랜덤 방에 배치

**요리 방법**:
1. 요리솥 접근
2. 상호작용 (E키)
3. 레시피 선택
4. MP 15 소모
5. 요리 완성

### 초보자 추천 요리

**1. 베리 주스**
- 재료: 베리 ×4
- 효과: MP +40, BRV +15
- 난이도: ⭐ (매우 쉬움)
- 입수: 베리 덤불

**2. 야채 샐러드**
- 재료: 채소 ×3, 허브 ×1
- 효과: HP +45, MP +25
- 난이도: ⭐
- 입수: 허브 식물

**3. 버섯 스튜**
- 재료: 버섯 ×3
- 효과: HP +50, BRV +20
- 난이도: ⭐
- 입수: 버섯 패치

**4. 생선 구이**
- 재료: 생선 ×1, 허브 ×1
- 효과: HP +60, 버프: 방어력 증가
- 난이도: ⭐⭐
- 입수: 물가, 허브 식물

### 요리 타이밍

**전투 전**:
- 버프 요리 (공격력/방어력 증가)
- 예: 불고기, 김치찌개

**전투 후**:
- 회복 요리 (HP/MP 회복)
- 예: 비빔밥, 라멘

**보스전 직전**:
- 최고급 요리
- 예: 드래곤 고기 바비큐

## ❓ 자주 묻는 질문

### Q1: BRV와 HP 공격 차이가 뭔가요?

**A**:
- **BRV 공격**: BRV만 축적, HP 데미지 없음
- **HP 공격**: 축적된 BRV로 HP 데미지, BRV 초기화

**언제 사용?**:
- BRV 낮을 때: BRV 공격
- BRV 충분할 때: HP 공격

---

### Q2: BREAK가 뭔가요?

**A**: 적의 BRV를 정확히 0으로 만들면 발동

**효과**:
- 보너스 데미지 +50%
- 적 1턴 스턴
- 자신 BRV +20%

---

### Q3: 요리솥이 안 보여요

**A**: 2025-11-15 업데이트로 층마다 최소 1개 보장!

**찾는 방법**:
- 모든 방 탐험
- Ω 심볼 찾기
- 미니맵 확인

---

### Q4: 어떤 패시브를 선택해야 하나요?

**A**: 초보자 추천 패시브 (코스트 15 이내)

1. **HP 증폭** (코스트 3): 최대 HP +25%
2. **신속** (코스트 2): 속도 +12%
3. **치명타 강화** (코스트 3): 크리티컬 확률 +15%
4. **마나 연결** (코스트 4): MP 재생 5/턴

**조합 예시**:
- 전사: HP 증폭 + 신속 + 치명타 강화 (코스트 8)
- 성직자: 마나 연결 + 치유 강화 + HP 증폭 (코스트 10)

---
