## 🗺️ 던전 시스템

### 절차적 생성

**BSP (Binary Space Partitioning) 알고리즘**:
- 매 층마다 새로운 맵 생성
- 랜덤 방 배치 (8~15개)
- 복도로 방 연결
- 특수 오브젝트 배치

**맵 크기**:
- 너비: 80 타일
- 높이: 50 타일
- 방 크기: 5x5 ~ 12x12

---

## 🏰 맵 구조

### 타일 종류

**FLOOR (바닥)**: `.`
- 이동 가능
- 안전 지대

**WALL (벽)**: `#`
- 이동 불가
- 시야 차단

**DOOR (문)**: `+`
- 통과 가능
- 잠금 가능 (열쇠 필요)

**STAIRS_UP (올라가는 계단)**: `<`
- 이전 층으로 이동
- 첫 번째 방에 배치

**STAIRS_DOWN (내려가는 계단)**: `>`
- 다음 층으로 이동
- 마지막 방에 배치

## 🎁 특수 오브젝트

### 1. 보물상자 (□)

**개수**: 층당 1~5개

**내용물**:
- 장비 (무기, 방어구, 장신구)
- 소비 아이템
- 골드
- 희귀 재료

**등급**:
- 일반: 70%
- 언커먼: 20%
- 레어: 8%
- 에픽: 2%

**잠금**:
- 20% 확률로 잠금
- 열쇠 또는 자물쇠 따기 필요

---

### 2. 떨어진 아이템 (*)

**개수**: 층당 3~15개

**종류**:
- 포션 (HP/MP 회복)
- 장비 (낮은 등급)
- 골드
- 재료

**위치**:
- 방 안
- 복도
- 랜덤 배치

---

### 3. 함정 (^)

**개수**: 층당 2~10개

**종류**:

**스파이크 함정**:
- 데미지: 5~20 HP
- 감지 가능 (지각 스탯)

**독 함정**:
- 독 상태 3턴
- HP 3%/턴 손실

**마법 함정**:
- 랜덤 디버프
- 침묵, 슬로우 등

**회피**:
- 지각 스탯으로 감지
- 회피 스탯으로 피하기
- 해제 스킬 사용

---

### 4. 채집 오브젝트

**2025-11-15 업데이트**: 스폰율 대폭 증가!

**개수**: 층당 **12~20개** (이전: 8~15개)

**종류**:
- 베리 덤불 (♣): 베리 3~6개
- 버섯 패치 (♠): 버섯 3~6개
- 허브 식물 (♦): 채소 3~7개
- 나무 (♣): 과일 + 나뭇가지 3~7개
- 물가 (≈): 물고기 3~7개
- 시체 (☠): 고기 3~6개
- 바위 (◙): 얼음 2~4개

**상세 정보**: [채집 시스템 가이드](gathering-system.md)

---

### 5. 요리솥 (Ω)

**2025-11-15 업데이트**: 층마다 최소 1개 보장!

**배치**:
- **층마다 무조건 1개** (100%)
- 20% 확률로 추가 1개 (최대 2개)

**기능**:
- 요리 제작
- MP 15 소모
- 52개 레시피

**상세 정보**: [요리 시스템 가이드](cooking.md)

---

### 6. 열쇠와 잠긴 문

**열쇠 (K)**:
- 층당 0~3개
- 특정 문 해금

**잠긴 문 (D)**:
- 복도에 배치
- 해당 열쇠 필요

**메커니즘**:
```
열쇠 획득 → 잠긴 문 도달 → 해금 → 다음 구역 진입
```

---

### 7. 텔레포터 (T)

**개수**: 5층마다 0~4쌍

**기능**:
- 즉시 이동
- 쌍으로 연결
- 양방향 이동

**활용**:
- 빠른 이동
- 탐험 효율 증가
- 전략적 후퇴

---

### 8. 용암 (L)

**출현**: 3층부터

**효과**:
- 진입 시 즉시 데미지
- 10~30 HP 손실/턴
- 화염 저항으로 감소

**위치**:
- 방 가장자리
- 일부 복도

**회피**:
- 우회 경로 찾기
- 화염 저항 장비
- 비행 스킬

---

### 9. 치유의 샘 (H)

**출현**: 30% 확률

**효과**:
- HP 100% 회복
- MP 100% 회복
- 상처 완전 치료
- 디버프 제거

**제한**:
- 층당 1회만 사용 가능

**위치**:
- 랜덤 방 중앙

---

### 10. 보스룸 경고 (!)

**출현**: 5층마다

**효과**:
- 보스룸 입구 표시
- 경고 메시지

**준비**:
- HP/MP 회복
- 버프 요리
- 장비 점검

---

## 👹 보스룸

### 보스 출현

**주기**: 5층마다 (5, 10, 15, 20...)

**위치**: 마지막 방

**경고**: 입구에 경고 타일 (!)

### 보스 특징

**강화된 스탯**:
- HP: 일반 적의 3~5배
- 공격력: 1.5배
- 방어력: 1.5배
- 특수 스킬

**패턴**:
- 페이즈 변화 (HP 50% 이하)
- 광역 공격
- 소환 (작은 적들)

**보상**:
- 보스 전용 장비
- 대량 골드
- 별조각 50~200개
- 희귀 재료

### 보스 전략

**준비**:
1. HP/MP 100% 회복
2. 버프 요리 섭취
3. 장비 최적화
4. 소비 아이템 준비

**전투**:
1. BRV 충분히 축적
2. BREAK 노리기
3. 패턴 파악
4. 회복 아이템 활용

**패턴 대응**:
- 광역 공격: 분산 배치
- 소환: 작은 적 먼저 처치
- 버프: 디스펠 사용

## 🎯 탐험 팁

### 효율적인 탐험 순서

**1단계: 맵 파악**
```
계단 확인 → 모든 방 위치 파악 → 특수 오브젝트 표시
```

**2단계: 채집**
```
채집 오브젝트 모두 수집 → 인벤토리 관리
```

**3단계: 전투**
```
적 조우 → 전투 → 경험치 획득
```

**4단계: 휴식**
```
안전 지대 찾기 → 휴식 → HP/MP 회복
```

**5단계: 요리**
```
요리솥 찾기 → 요리 제작 → 버프 적용
```

**6단계: 다음 층**
```
준비 완료 → 내려가는 계단 → 다음 층
```

---

### 안전 탐험 팁

**1. 시야 관리**
- 미니맵 활용
- 밝은 곳에서 전투
- 어둠 주의

**2. 인벤토리 관리**
- 무게 제한 주의
- 불필요한 아이템 버리기
- 정리 습관

**3. HP/MP 관리**
- 전투 전 회복
- 포션 비축
- 휴식 활용

**4. 함정 주의**
- 지각 스탯 투자
- 천천히 이동
- 해제 스킬 습득

**5. 자원 절약**
- MP 효율적 사용
- 포션 남용 금지
- 요리로 버프

---

### 층별 탐험 전략

**초반 (1~5층)**:
- 모든 방 탐험
- 채집 최우선
- 경험치 파밍

**중반 (6~10층)**:
- 효율적인 루트
- 보스 준비
- 장비 업그레이드

**후반 (11층~)**:
- 선택적 전투
- 생존 우선
- 빠른 클리어

---

### 숨겨진 방 찾기

**징후**:
- 벽 뒤 공간
- 이상한 벽 패턴
- 미니맵 불일치

**발견 방법**:
- 벽 조사
- 폭발 스킬
- 탐지 스킬

**보상**:
- 희귀 장비
- 대량 골드
- 특별 아이템

## 💡 고급 탐험 테크닉

### 1. 루트 최적화

**최단 경로**:
- 불필요한 방 스킵
- 직선 루트
- 빠른 클리어

**완전 탐험**:
- 모든 방 탐험
- 모든 오브젝트 수집
- 경험치 최대화

### 2. 위험 회피

**강한 적 회피**:
- 레벨 차이 확인
- 우회 경로 찾기
- 스텔스 스킬

**함정 패턴 파악**:
- 복도에 많음
- 좁은 통로 주의
- 회피 동선

### 3. 자원 관리

**포션 사용 타이밍**:
- HP 50% 이하
- 보스전 직전
- 위급 상황만

**MP 효율**:
- 일반 전투: 기본 공격
- 위험 전투: 스킬 사용
- 보스전: 궁극기

### 4. 경험치 파밍

**효율적인 레벨링**:
- 같은 레벨 적 반복
- 채집 + 전투 병행
- 보스 반복 (가능 시)
