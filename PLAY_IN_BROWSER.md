# 🌐 브라우저에서 Dawn of Stellar 플레이하기

> 설치 없이 브라우저에서 바로 게임을 플레이하는 방법

---

## 🎮 플레이 방법

### 1️⃣ GitHub Codespaces (추천!) ⭐

**가장 쉽고 빠른 방법입니다!**

#### 단계별 가이드:

1. **저장소 페이지 방문**
   - https://github.com/APTOL-7176/Dos

2. **Code 버튼 클릭**
   - 오른쪽 상단의 초록색 "Code" 버튼 클릭

3. **Codespaces 탭 선택**
   - "Codespaces" 탭 클릭 (Local, SSH 옆에 있음)

4. **Codespace 생성**
   - "Create codespace on main" 버튼 클릭
   - 잠시 기다리면 브라우저에서 VS Code가 열립니다

5. **게임 실행**
   - 하단 터미널에서 다음 명령어 입력:
   ```bash
   python main.py
   ```

6. **게임 즐기기!** 🎉
   - 방향키로 이동
   - 숫자 키로 스킬 사용
   - ESC로 메뉴 열기

#### Codespaces 장점:
- ✅ 설치 불필요
- ✅ 브라우저에서 바로 실행
- ✅ 무료 (월 60시간)
- ✅ 빠른 속도
- ✅ 자동 환경 설정

### 3️⃣ 모바일에서 플레이 📱

**모바일 브라우저에서도 플레이 가능!**

Dawn of Stellar는 모바일 기기에서도 플레이할 수 있습니다. GitHub Codespaces와 Gitpod 모두 모바일 브라우저를 지원합니다.

#### 모바일 플레이 가이드:

**1. GitHub Codespaces (모바일)**
- ✅ iOS Safari, Chrome 지원
- ✅ Android Chrome, Samsung Internet 지원
- ✅ 터치스크린 키보드 자동 표시
- ⚠️ 가로 모드 권장 (화면 공간 확보)

**모바일에서 시작하기:**
1. 모바일 브라우저에서 GitHub 저장소 접속
2. "Code" 버튼 → "Codespaces" 탭
3. "Create codespace on main" 탭
4. VS Code Web 환경 로딩 대기
5. 터미널에서 `python main.py` 입력

**2. Gitpod (모바일)**
- ✅ 모바일 최적화 UI
- ✅ 터치 제스처 지원
- ✅ 빠른 로딩

**모바일 링크:**
- [Gitpod에서 열기](https://gitpod.io/#https://github.com/APTOL-7176/Dos)

#### 모바일 조작 팁:

**키보드 입력:**
- 화면 탭 → 소프트 키보드 표시
- 방향키: 키보드의 화살표 키 사용
- 숫자: 키보드 상단의 1~6 키 사용
- 문자: WASD, QERS 등 일반 키보드 입력

**화면 설정:**
- 📱 **가로 모드 필수**: 게임 화면이 크므로 가로 모드 사용
- 🔍 **확대/축소**: 핀치 제스처로 텍스트 크기 조절
- 📏 **전체 화면**: 브라우저 전체 화면 모드 활성화 권장

**성능 최적화:**
- Wi-Fi 연결 권장 (데이터 사용량 고려)
- 백그라운드 앱 종료 (메모리 확보)
- 브라우저 캐시 정리
- 배터리 세이버 모드 해제

#### 모바일 제한사항:

⚠️ **알아두세요:**
- 터미널 기반 게임이므로 터치 인터페이스가 아닌 키보드 입력 방식
- 작은 화면에서는 텍스트가 작게 보일 수 있음
- 일부 특수 키 조합은 모바일에서 입력 어려움
- 블루투스 키보드 연결 시 더 편리한 플레이 가능

#### 추천 환경:

**최상:**
- 태블릿 (iPad, Galaxy Tab 등) + 블루투스 키보드
- 10인치 이상 화면
- 가로 모드
- Wi-Fi 연결

**권장:**
- 스마트폰 (6인치 이상)
- 가로 모드
- 소프트 키보드

**최소:**

## 🎯 게임 조작법

### 메뉴 탐색:
- **방향키**: 메뉴 이동
- **Enter**: 선택
- **ESC**: 뒤로 가기 / 메뉴 열기

### 던전 탐험:
- **방향키 / WASD**: 이동
- **Shift + 방향키**: 대시

### 전투:
- **1~6**: 스킬 선택
- **Q**: BRV 공격
- **E**: HP 공격
- **R**: 궁극기
- **Space**: 아이템 사용

### 기타:
- **I**: 인벤토리
- **C**: 캐릭터 정보
- **M**: 맵
- **ESC**: 메뉴

## 🐛 문제 해결

### Q: Codespace가 시작되지 않아요
**A**:
1. 브라우저 새로고침 (Ctrl + F5)
2. 다른 브라우저 시도
3. GitHub 로그인 상태 확인

### Q: 게임이 실행되지 않아요
**A**:
```bash
# 의존성 재설치
pip install -r requirements.txt

# 게임 실행
python main.py
```

### Q: 화면이 깨져 보여요
**A**:
1. 터미널 크기 조정 (최소 80x50)
2. 폰트 크기 조정
3. 전체 화면 모드 사용

### Q: 한글이 깨져요
**A**:
```bash
# UTF-8 설정 확인
export LANG=ko_KR.UTF-8
export LC_ALL=ko_KR.UTF-8

# 게임 실행
python main.py
```

### 📱 모바일 전용 문제 해결

#### Q: 모바일에서 키보드가 안 나타나요
**A**:
1. 터미널 영역을 탭하여 포커스 설정
2. 화면 하단의 키보드 아이콘 탭
3. 브라우저 설정에서 "데스크톱 사이트 요청" 해제
4. 다른 브라우저 시도 (Chrome → Safari, 또는 그 반대)

#### Q: 모바일에서 화면이 너무 작아요
**A**:
1. **가로 모드로 전환** (필수)
2. 브라우저 확대 (핀치 제스처)
3. VS Code 설정 → 터미널 폰트 크기 증가
4. 전체 화면 모드 활성화
5. 태블릿 사용 권장 (더 큰 화면)

#### Q: 모바일에서 방향키를 어떻게 누르나요?
**A**:
1. **소프트 키보드 사용**:
   - iOS: 화면 탭 → 키보드 표시 → 방향키 아이콘
   - Android: 화면 탭 → 키보드 표시 → 방향키
2. **대체 조작**:
   - WASD 키 사용 (W=위, A=왼쪽, S=아래, D=오른쪽)
   - 숫자 패드 사용 (일부 키보드)
3. **블루투스 키보드 연결** (권장):
   - 설정 → 블루투스 → 키보드 페어링
   - 물리 키보드로 편안한 플레이

#### Q: 모바일에서 너무 느려요
**A**:
1. Wi-Fi 연결 사용 (4G/5G보다 안정적)
2. 백그라운드 앱 모두 종료
3. 브라우저 캐시 및 쿠키 삭제
4. 기기 재시작
5. 저사양 기기는 성능 제한 있을 수 있음

#### Q: 모바일에서 ESC 키를 누를 수 없어요
**A**:
1. iOS: Cmd + . (점) 또는 소프트 키보드에서 ESC 키 찾기
2. Android: 뒤로 가기 버튼 또는 소프트 키보드 ESC
3. 일부 키보드 앱은 ESC 키가 숨겨져 있을 수 있음
4. 블루투스 키보드 사용 권장

#### Q: 모바일에서 복사/붙여넣기가 안 돼요
**A**:
- 터미널에서 텍스트 길게 누르기 → 복사/붙여넣기 메뉴
- 일부 모바일 브라우저는 터미널 복사/붙여넣기 제한됨
- Gitpod이 Codespaces보다 모바일 친화적

#### 💡 모바일 플레이 최적화 팁

**최고의 모바일 경험을 위한 설정:**
1. ✅ **가로 모드 사용** (필수!)
2. ✅ **전체 화면 모드** 활성화
3. ✅ **Wi-Fi 연결** (안정성)
4. ✅ **블루투스 키보드** 연결 (선택, 강력 추천)
5. ✅ **배터리 충분** (게임 플레이 중 충전 권장)
6. ✅ **방해 금지 모드** (알림 끄기)

**태블릿 사용자:**
